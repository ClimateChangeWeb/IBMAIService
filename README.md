# IBM AI Service

This project is the IBM AI server for the climate change website.

# Installation

Make a .env file in the root like:

```
MONGODB_URI=
DISCOVER_API_KEY=
DISCOVER_URL=
TEXT_TO_SPEECH_API_KEY=
TEXT_TO_SPEECH_URI=
```

Then running

```bash
npm i
```

and

```
npm start
```

to running the server.
